# 小動物鑑別診斷輔助系統 — 專業評估報告

**評估者角色：** 資深小動物內科獸醫師（具內科、心臟科、急診經驗）  
**評估日期：** 2026-02-02  
**參考架構：** Gough & Murphy (2015)、Thompson (2024) 4th Ed、BSAVA Neurology Manual

---

## 一、整體使用體驗與模擬臨床情境測試

### 模擬情境 1：急診 — 8歲未絕育母約克夏，PU/PD + 陰道分泌物 + 發燒

我在左側點選「多尿/多渴」，展開後可看到 12 項鑑別診斷，內容涵蓋常見的糖尿病、CKD、Cushing's 到少見的尿崩症。隨後切換至「多症狀交叉比對」，選入「多尿/多渴」+「體溫過高/發燒」，系統正確地列出子宮蓄膿作為交叉結果。

**正面評價：**
- 交叉比對的邏輯正確，確實能將 Pyometra 列為同時符合多個症狀的疾病
- 每張疾病卡片的中英文對照有助學習
- 頻率標籤（常見/少見/罕見）符合臨床經驗

**發現問題：**
- 無法加入「陰道分泌物」做三症狀交叉（因為這個症狀在側邊欄需要滾動很遠才找到）
- 沒有「腹痛」這個獨立症狀可選，而腹痛是子宮蓄膿的關鍵主訴之一
- 缺乏**緊急度標示** — 閉鎖型子宮蓄膿是外科急診，但系統無法凸顯這一點

### 模擬情境 2：神經科 — 10歲柯基，突發頭歪一邊 + 走路歪斜

我搜尋「頭歪」、「head tilt」 — **找不到任何結果**。這是一個重大缺失。Head tilt 是臨床上極為常見的神經科主訴，飼主描述「狗的頭一直歪一邊」可能是獸醫門診中最常見的神經急診之一。

退而求其次，我選擇「共濟失調」，確實找到前庭疾病作為鑑別之一。但在真實臨床場景中，飼主不會用「共濟失調」這個詞 — 他們會說「頭歪了」或「一直轉圈」。

### 模擬情境 3：內科 — 5歲金毛，黏膜蒼白 + 虛弱

搜尋「蒼白」、「pallor」 — **找不到**。蒼白是理學檢查中最重要的發現之一，直接指向貧血、休克或血管收縮等嚴重問題。系統中沒有「蒼白/發紺」作為獨立可查詢的項目。

### 模擬情境 4：急診 — 3歲法鬥，急性腹部劇痛

搜尋「腹痛」、「abdominal pain」 — **找不到**。急性腹痛是最常見的急診主訴之一（胰臟炎、GDV、腸阻塞、腹膜炎、子宮蓄膿），但程式中沒有這個項目。「腹部膨大」不等於「腹痛」，這是兩個完全不同的臨床表現。

## 二、使用者介面與互動設計評估

### 優點（值得保留）

1. **配色方案** — 綠色系醫療風格專業、清爽，不會造成視覺疲勞
2. **卡片展開設計** — 點擊展開詳細資訊的設計節省空間，避免資訊過載
3. **中英雙語** — 對華語區獸醫查閱英文文獻非常有幫助
4. **症狀描述方塊** — 每個症狀頂部的綠色描述框提供簡潔的臨床定義
5. **分類標籤系統** — 按病因分類（內分泌、消化、感染等）有助臨床思維
6. **多症狀交叉比對** — 核心功能正確且實用，符合臨床推理流程
7. **物種篩選** — 犬/貓分開是必要功能
8. **行動裝置響應式** — 支援手機使用，適合臨床現場查詢

### 需改進之處

#### 2.1 導航效率問題

- **左側症狀列表過長**：48個症狀需要大量滾動。應考慮加入**摺疊式分類**（點擊「消化道」展開該系統的所有症狀），或加入**字母索引跳轉**
- **搜尋匹配不夠智能**：搜尋「拉肚子」找不到「腹瀉」，搜尋「吐」找不到「嘔吐」。建議加入**同義詞對照表**（俗語 → 醫學術語）
- **行動裝置上側邊欄佔據太多空間**：在手機上，250px 的側邊欄壓縮了主要內容區域，建議改為**抽屜式選單**

#### 2.2 資訊層次不足

- **所有鑑別診斷同等呈現**：常見病與罕見病在視覺上差異不大。建議用**更明顯的視覺層級**區分 — 例如常見疾病卡片預設略微展開，或用不同底色
- **缺少檢查分層**：「建議檢查」項目列表中混合了基礎檢查（CBC、X光）和進階檢查（MRI、CT）。在基層動物醫院中，應先列出門診可做的基本檢查
- **無「排除優先順序」**：當鑑別清單有 10+ 個疾病時，獸醫師需要知道「先排除什麼」

#### 2.3 缺少的關鍵功能

| 功能 | 重要性 | 說明 |
|------|--------|------|
| 緊急度標示 | ⭐⭐⭐⭐⭐ | GDV、尿道阻塞、氣胸等需立即標紅 |
| 年齡篩選 | ⭐⭐⭐⭐⭐ | 幼年 vs 成年 vs 老年的鑑別完全不同 |
| 品種傾向 | ⭐⭐⭐⭐ | 臘腸犬→IVDD、拳師犬→心臟病等 |
| 儲存/列印 | ⭐⭐⭐⭐ | 能列印或匯出 PDF 供病歷附件使用 |
| 排除功能 | ⭐⭐⭐⭐ | 「已排除 X 檢查」→ 灰掉對應疾病 |
| 反向查詢 | ⭐⭐⭐ | 輸入「ALT 升高」→ 列出可能疾病 |
| 離線使用 | ⭐⭐⭐ | PWA 支援，手術室/病房無網路時可用 |
| 相關症狀提示 | ⭐⭐⭐ | 選「嘔吐」後提示：「是否也有腹瀉？腹痛？」 |

## 三、重要缺失症狀 — 依臨床優先順序排列

以下是目前程式中**完全缺失**但在臨床上極為常見的症狀/理學檢查發現。依據 Thompson (2024) Small Animal Medical Differential Diagnosis 4th Edition 的完整目錄、Gough & Murphy (2015) 架構，以及我的臨床經驗進行交叉比對。

### A. 最高優先級 — 每日門診/急診都會遇到

#### 1. 急性腹痛 (Acute Abdominal Pain)
- **缺失嚴重度：極高**
- 主要鑑別：胰臟炎、腸阻塞/異物、GDV、腹膜炎、子宮蓄膿、脾臟扭轉、尿道阻塞、腸套疊
- 臨床意義：急診最常見主訴之一，許多是外科急診

#### 2. 頭歪斜 (Head Tilt)
- **缺失嚴重度：極高**
- 主要鑑別：
  - 周邊型前庭：特發性前庭症候群（老犬/貓最常見）、中內耳炎、鼻咽息肉（貓）、甲狀腺低下（犬）
  - 中樞型前庭：腦幹腫瘤、腦炎（GME/NME）、腦血管事件、Metronidazole 中毒
- 臨床意義：飼主最常描述的神經症狀，需快速區分周邊 vs 中樞

#### 3. 蒼白 (Pallor) / 發紺 (Cyanosis)
- **缺失嚴重度：極高**
- 蒼白鑑別：貧血（IMHA、出血、骨髓疾病）、休克、血管收縮
- 發紺鑑別：呼吸衰竭、右至左心臟分流、高鐵血紅素血症、低心輸出量
- 臨床意義：理學檢查第一個注意到的異常，直接影響急救優先級

#### 4. 眼球震顫 (Nystagmus)
- **缺失嚴重度：高**
- 鑑別方向取決於震顫類型：水平型/旋轉型（周邊）vs 垂直型/方向變化型（中樞）
- 與 Head Tilt 常同時出現，可獨立列出或合併為「前庭症候群」

#### 5. 喘鳴 / 吸氣音異常 (Stertor / Stridor)
- **缺失嚴重度：高**
- Stertor（鼻咽部）：鼻咽息肉、短頭犬、腫瘤
- Stridor（喉部）：喉頭麻痺、喉部水腫（過敏反應）、喉部腫瘤、氣管塌陷
- 臨床意義：與「呼吸困難」不同，定位上呼吸道阻塞

#### 6. 瘀斑 / 點狀出血 (Petechiae / Ecchymoses)
- **缺失嚴重度：高**
- 主要鑑別：IMTP（免疫介導性血小板減少）、DIC、Ehrlichia/Anaplasma、骨髓疾病、抗凝血劑中毒、血管炎
- 臨床意義：直接指向凝血/血小板異常，需要緊急處理

#### 7. 口臭 (Halitosis) / 口炎 (Stomatitis)
- **缺失嚴重度：中高**
- 口臭鑑別：牙周病（最常見）、口腔腫瘤、腎衰竭（尿毒味）、糖尿病酮酸中毒（水果味）、口炎
- 口炎鑑別：貓慢性齦口炎（FCV 相關）、腎衰竭性口炎、免疫介導性
- 臨床意義：飼主常見主訴，且可提示全身性疾病

#### 8. 水腫 (Peripheral Oedema)
- **缺失嚴重度：中高**
- 鑑別：低白蛋白血症（PLE/PLN/肝病）、右心衰竭、淋巴管阻塞、局部感染/發炎、血管炎、蛇咬
- 臨床意義：水腫與腹水的鑑別方向重疊但不完全相同

#### 9. 外耳炎 (Otitis Externa)
- **缺失嚴重度：中高**
- 主要鑑別：過敏（異位性皮膚炎最常見）、耳疥蟲、異物、細菌/酵母菌感染、耳道息肉/腫瘤
- 臨床意義：小動物門診最常見主訴之一，反覆外耳炎常指向潛在過敏問題

#### 10. 心搏過速 / 心搏過緩 (Tachycardia / Bradycardia)
- **缺失嚴重度：中高**
- 心搏過速：疼痛、發燒、貧血、心衰竭、甲亢、Pheochromocytoma
- 心搏過緩：高血鉀（Addison's）、房室傳導阻滯、洋地黃中毒、顱內壓升高、甲低
- 臨床意義：心律不整的基本分類方式

### B. 高優先級 — 每週都會遇到

#### 11. Horner Syndrome（交感神經損傷）
- 縮瞳 + 眼球內陷 + 第三眼瞼突出 + 上眼瞼下垂
- 鑑別：中內耳炎、胸腔腫瘤/頸部損傷、特發性（最常見）

#### 12. 少尿 / 無尿 (Oliguria / Anuria)
- 鑑別：急性腎損傷、尿道阻塞（公貓急診！）、腎後性阻塞、休克
- 與「頻尿」是完全不同的臨床意義

#### 13. 顏面神經麻痺 (Facial Nerve Paralysis)
- 鑑別：特發性、中內耳炎、甲狀腺低下、多發性神經炎、腫瘤
- 臨床表現：無法閉眼、唇下垂、耳下垂

#### 14. 肌肉萎縮 / 惡病質 (Cachexia / Muscle Wasting)
- 鑑別：惡性腫瘤、心臟病、CKD、甲亢、庫欣（選擇性肌肉萎縮）
- 與「體重減輕」的鑑別方向不同 — 著重蛋白質代謝異常

#### 15. 皮膚腫塊 (Cutaneous Masses / Skin Lumps)
- 鑑別：肥大細胞瘤、脂肪瘤、組織球瘤、皮脂腺腫瘤、膿瘍
- 臨床意義：飼主帶寵物就診最常見的原因之一

### C. 中優先級 — 理學檢查發現/實驗室異常

#### 16. 脾腫大 (Splenomegaly)
#### 17. 肝腫大 (Hepatomegaly)
#### 18. 腎腫大 (Renomegaly)
#### 19. 過敏反應 / 蕁麻疹 (Anaphylaxis / Urticaria)
#### 20. 尿液顏色異常 (Discoloured Urine)
#### 21. 耳聾 (Deafness)
#### 22. 心包膜積液 (Pericardial Effusion)
#### 23. 胸腔積液 (Pleural Effusion) — 已在呼吸困難下列出，但應獨立
#### 24. 多飲但非多尿 (Polydipsia without Polyuria)
#### 25. 吞氣 / 腹脹 (Aerophagia / Gastric Dilatation)

### D. 建議新增的全新分類：實驗室數據異常

這是程式目前完全缺少的一個維度。獸醫師拿到血檢報告後，常需要反向查詢「這個數值異常代表什麼？」

| 異常項目 | 優先級 | 說明 |
|----------|--------|------|
| 貧血 (Anaemia) — 再生性 vs 非再生性 | ⭐⭐⭐⭐⭐ | 最常見血液學異常 |
| 血小板減少 (Thrombocytopenia) | ⭐⭐⭐⭐⭐ | IMTP、Ehrlichia、DIC |
| 肝指數升高 (Elevated ALT/ALP/GGT) | ⭐⭐⭐⭐⭐ | 每日都在判讀 |
| 腎指數升高 (Azotaemia) | ⭐⭐⭐⭐⭐ | 腎前/腎性/腎後鑑別 |
| 高/低血糖 | ⭐⭐⭐⭐ | 胰島素瘤 vs DM vs 敗血症 |
| 高/低血鈣 | ⭐⭐⭐⭐ | 淋巴瘤、副甲狀腺、CKD |
| 高/低血鉀 | ⭐⭐⭐⭐ | Addison's、CKD、尿道阻塞 |
| 高/低血鈉 | ⭐⭐⭐ | Addison's、SIADH |
| 白血球增多/減少 | ⭐⭐⭐ | 感染、壓力、骨髓疾病 |

## 四、具體功能改進建議

### 建議 1：新增「緊急度」標籤系統

```
🔴 Emergency（需立即處理，分鐘內）：
   GDV、尿道阻塞（公貓）、氣胸（張力性）、癲癇重積、嚴重過敏反應

🟠 Urgent（需數小時內處理）：
   子宮蓄膿（閉鎖型）、IMHA 急性、急性腎損傷、DKA

🟡 Semi-urgent（24小時內）：
   急性腹痛、嚴重呼吸困難、FLUTD 非阻塞型

🟢 Non-urgent（可安排門診）：
   慢性皮膚搔癢、慢性體重減輕、慢性腹瀉
```

### 建議 2：增加年齡/品種篩選維度

在物種篩選旁增加：
- **年齡範圍**：幼年（<1歲）、青年（1-3歲）、成年（3-7歲）、老年（>7歲）
- **品種傾向**：每個鑑別診斷標註好發品種
  - 例：IVDD → 臘腸犬、法鬥、比格
  - 例：MMVD → 查理士王小獵犬、馬爾濟斯、吉娃娃
  - 例：DCM → 杜賓、大丹、拳師

### 建議 3：檢查項目分層

將「建議檢查」分為兩層：

**第一線（基層門診可做）：**
- 理學檢查、CBC、基礎生化、尿液分析、X光、眼壓、血壓

**第二線（需轉診或特殊設備）：**
- 超音波、CT/MRI、內視鏡、特殊血液檢測（ACTH stim、膽汁酸）、腦脊髓液、切片

### 建議 4：新增「實驗室數據查詢」Tab

在現有「單一症狀查詢」和「多症狀交叉比對」旁，新增第三個 Tab：
- 使用者選擇異常的血檢/尿檢數值
- 系統反向列出可能的疾病
- 例如：選擇「ALT 升高 + 黃疸」→ 肝炎、膽管肝炎、膽道阻塞、肝脂肪沉積（貓）

### 建議 5：智能提示 — 「相關症狀」建議

當使用者選擇一個症狀後，系統可提示：
- 「您選擇了『嘔吐』，請問是否同時觀察到：腹瀉？腹痛？黑便？體重減輕？」
- 這模擬了獸醫問診的流程，引導使用者更完整地收集病史

### 建議 6：飼主描述 → 醫學術語的搜尋對照

建立一個俗語對照表用於搜尋增強：

| 飼主可能說的話 | 對應醫學術語 |
|-------------|------------|
| 拉肚子、軟便 | 腹瀉 |
| 吐、嘔 | 嘔吐 |
| 喝很多水 | 多渴/多飲 |
| 尿很多 | 多尿 |
| 頭歪一邊 | 頭歪斜 |
| 一直轉圈 | 共濟失調/前庭症候群 |
| 抓癢、一直舔 | 搔癢 |
| 腳痛、走路怪怪的 | 跛行 |
| 肚子脹大 | 腹部膨大 |
| 眼睛白白的 | 白內障/失明 |
| 抽筋、倒地 | 癲癇/昏厥 |
| 沒精神、懶懶的 | 虛弱/嗜睡 |

## 五、資料庫架構改進建議

### 現狀問題
目前所有數據硬編碼在 JavaScript 物件中（約 1700 行），不利於維護和擴充。

### 建議架構

```
/data
  ├── symptoms.json          ← 症狀定義（ID、名稱、分類）
  ├── differentials.json     ← 疾病資料庫（名稱、描述、頻率、物種、品種傾向）
  ├── symptom-disease.json   ← 症狀-疾病對應關係
  ├── workups.json           ← 檢查項目（分層、侵入性、成本等級）
  ├── lab-abnormalities.json ← 實驗室異常（新增分類）
  └── synonyms.json          ← 搜尋同義詞對照
```

這樣的結構允許：
- 獨立更新某個疾病的資訊而不影響其他部分
- 社群貢獻新的鑑別診斷
- 多語言擴充
- API 化後供其他應用使用

## 七、總結

### 目前程式的完成度評估

| 面向 | 評分 (1-10) | 說明 |
|------|-------------|------|
| 介面設計 | 8/10 | 清晰專業，配色舒適 |

| 症狀涵蓋度 | 5/10 | 48 個症狀，缺少約 30+ 個臨床常見項目 |
| 臨床實用性 | 6/10 | 基本可用，但缺少急性腹痛、頭歪斜等關鍵項目 |
| 互動功能 | 5/10 | 缺少儲存、排除、反向查詢等進階功能 |
| 行動裝置體驗 | 7/10 | 響應式設計已有，但側邊欄需優化 |

### 最優先的三項改進

1. **新增缺失的關鍵症狀**（腹痛、頭歪斜、蒼白/發紺、瘀斑、口炎、外耳炎、喘鳴）— 這些每天門診都會遇到
2. **加入緊急度標示** — 臨床上最重要的資訊就是「這個需要多快處理」
3. **建立實驗室數據查詢功能** — 獸醫師拿到血檢報告後的第一件事就是查異常值的鑑別

此工具的核心概念和基礎架構都很好，只需在**廣度**（更多症狀）和**深度**（更多臨床決策支援功能）兩個維度擴充，就能成為臨床獸醫師真正日常使用的實用工具。
